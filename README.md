# 💻 Cyber Simulator

**Cyber Simulator** adalah simulasi interaktif berbasis web yang menggambarkan aktivitas dunia cyber seperti hacking simulation, sistem terminal, dan visualisasi keamanan digital.

Project ini dibuat sebagai **eksperimen visual dan edukasi** untuk menampilkan konsep dunia cybersecurity melalui tampilan yang futuristik dan interaktif.

---

## 🚀 Live Demo

🌐 https://agusadhitama.github.io/cyber-simulator/

---

## 📌 Deskripsi

Cyber Simulator menampilkan pengalaman simulasi dunia cyber melalui antarmuka yang menyerupai **terminal hacking dan sistem digital**.

Pengguna dapat melihat berbagai elemen visual seperti:

* Animasi jaringan digital
* Tampilan terminal cyber
* Efek futuristik ala sistem hacking
* Simulasi aktivitas cyber

Project ini dibuat untuk **eksplorasi UI/UX bertema cybersecurity** serta latihan pengembangan web menggunakan JavaScript.
